# 業種別の深掘り観点

Use this file to adapt questions and sections for common website types.

## 整体院・治療院

Key questions:

- どんな症状に強いですか？
- 保険診療、自費施術、どちらが中心ですか？
- 初回の流れと料金は明確ですか？
- 医療・治療効果の断定表現を避ける必要がありますか？

Useful sections:

- 症状別の悩み
- 施術方針
- 施術の流れ
- お客様の声
- アクセス
- 予約CTA

## 美容室・サロン

Key questions:

- 得意なスタイルや施術は何ですか？
- 新規客に来てほしいですか、リピート強化ですか？
- スタイリストの指名導線は必要ですか？
- 写真素材はありますか？

Useful sections:

- 得意なスタイル
- メニュー
- 施術事例
- スタッフ紹介
- 予約導線

## 士業・専門家

Key questions:

- 相談内容は何に特化していますか？
- 初回相談の有料・無料はどちらですか？
- 実績や対応事例は掲載できますか？
- 専門用語をどこまで噛み砕く必要がありますか？

Useful sections:

- よくある相談
- 対応業務
- 解決事例
- 相談の流れ
- プロフィール

## コンサル・講座

Key questions:

- 何を改善するサービスですか？
- 対象者の現在地とゴールは何ですか？
- 実績、受講者の声、事例はありますか？
- 無料相談、説明会、資料請求のどれに誘導しますか？

Useful sections:

- 課題提起
- 支援内容
- 得られる結果
- 実績
- カリキュラムまたは支援の流れ

## 飲食店・店舗

Key questions:

- 予約、来店、テイクアウト、どれが主目的ですか？
- 人気メニューや看板商品は何ですか？
- 利用シーンは何ですか？
- 営業時間、アクセス、決済方法は明確ですか？

Useful sections:

- コンセプト
- 人気メニュー
- 店内写真
- 利用シーン
- アクセス
- 予約CTA

## ポートフォリオ・個人サービス

Key questions:

- どんな仕事を受けたいですか？
- 何を得意領域として見せたいですか？
- 実績は成果・担当範囲・制作意図まで説明できますか？
- 問い合わせ前の不安は何ですか？

Useful sections:

- 得意領域
- 実績
- 対応範囲
- 制作の流れ
- 料金目安
- 問い合わせCTA
